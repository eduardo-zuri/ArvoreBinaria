# Árvore Binária Interativa

Uma aplicação web desenvolvida para oferecer uma visualização intuitiva e dinâmica do funcionamento de Árvores Binárias. Desenvolvida como uma atividade acadêmica.

## 🚀 Funcionalidades

- **Adicionar:** Insira qualquer caractere no campo. A aplicação construirá a árvore nó a nó visualmente.
- **Percorrer (Print):** Demonstra visualmente como as funções recursivas navegam pela estrutura binária.
- **Tamanho:** Exibe o número total de nós inseridos na árvore.
- **Altura da Árvore:** Retorna a altura total da estrutura.
- **Altura Específica:** Clique em qualquer nó individual para verificar a sua altura dentro da árvore.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação da página.
- **CSS3:** Estilização, animações (keyframes) e responsividade do layout.
- **JavaScript Vanilla:** Lógica de manipulação do DOM e algoritmos de estrutura de dados (Árvore Binária de Busca).
